# Lista de Tarefas

![Preview da Lista de Tarefas](lista%20de%20tarefas.png)

Este é um projeto de Lista de Tarefas desenvolvido com:

* HTML
* CSS
* JavaScript
* LocalStorage

## Funcionalidades

* Adicionar novas tarefas
* Marcar tarefas como concluídas
* Excluir tarefas
* Armazenamento automático no navegador
* Carregamento das tarefas salvas ao abrir a página
* Interface simples e intuitiva

## Objetivo

Projeto criado para praticar manipulação do DOM, lógica de programação e armazenamento de dados no navegador utilizando LocalStorage, além de compor meu portfólio profissional para oportunidades de estágio em Desenvolvimento Frontend.

## Como usar

Abra o arquivo:

index.html

no navegador

ou acesse a versão online pelo GitHub Pages.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6)
* Web Storage API
